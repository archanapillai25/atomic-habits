⚛️ Atomic Habits — v1.0

> Build sustainable habits. Stay focused. Stay hydrated. Stay healthy.

A lightweight, beautiful, self-hosted habit tracker designed for focus, wellness, and clarity.


## ✨ Features
- 💧 Hydration tracker with real-time progress bar  
- ⏱️ Pomodoro timer with countdown & alert  
- ✅ Task manager with checkbox toggles & strikethrough  
- 🚶 Stand-up reminder (configurable interval)  
- ⚙️ Settings page — customize goals & timers  
- 🌐 Responsive & modern gradient UI  

## 🚀 Quick Start

1. Install Docker Desktop
2. Open command prompt and Run this command
   docker run -d -p 8080:8080 archanapillai25/archanapillai25:latest
3. Open http://localhost:8080
4. To stop app
   docker stop 08aeeedf1877093140c481d1c18bbbd0d099ad507eb8c13f806f4b548add23c0
